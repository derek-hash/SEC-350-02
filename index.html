// 02 — Network Architecture

Firewall Configuration

01
edge01-derek // 10.0.17.110 / 172.16.50.2 / 172.16.150.2

Perimeter Firewall — WAN / DMZ / LAN

Zone-based policy with WAN, DMZ, and LAN zones. DNAT rules forward TCP 80/443 to nginx-derek and TCP 22 to jump-derek. ICMP from WAN to local is explicitly dropped. NAT masquerade on eth0 for outbound. Named power-adm local account for all configuration.

edge01
02
fw-mgmt-derek // 172.16.150.3 / 172.16.200.2

Management Firewall — LAN / MGMT

Separates LAN from the MGMT zone. Permits inbound SSH (22) and RDP (3389) from LAN to MGMT. All other traffic default-dropped. Global state policy allows established and related return traffic. Named power-adm local account for all configuration.

fw-mgmt

mgmt02-derek // 172.16.200.11

Domain Controller + CA + DNS — derek.local

Windows Server promoted to AD DS Domain Controller for derek.local. Enterprise Root CA (derek-CA) issues certs for HTTPS and SSH. DNS A records configured for all hosts except traveler. power-adm domain account used for all post-promotion configuration.

// 03 — Security Controls

WAF + PKI + SIEM

WAF
ModSecurity
Web Application Firewall

ModSecurity deployed on nginx-derek with SecRuleEngine set to On. Protects against SQLi, XSS, path traversal, and command injection. nginx has no outbound internet access — confirmed via firewall and route check. HTTPS served via cert from derek-CA.

PKI
Enterprise CA
derek-CA Root Certificate

Enterprise Root CA on mgmt02-derek. Signed certs distributed to nginx for HTTPS and jump for SSH CA auth. CA root PEM installed on all Linux hosts via /usr/local/share/ca-certificates. Passwordless SSH to jump-derek uses TrustedUserCAKeys.

LOG
Wazuh SIEM
Centralized Log Aggregation

All DMZ, LAN, and MGMT hosts forward logs to Wazuh at 172.16.200.10. Wazuh agents on all Linux hosts. Windows event forwarding on mgmt02-derek and wks01-derek. MITRE ATT&CK mapping enabled with full alert correlation.

jump-derek // 172.16.50.4

Named Guest Account — derek-guest

Created via useradd. Default shell /bin/bash. Default champuser account locked — passwd -l and nologin shell applied. Passwordless SSH from traveler using certificate signed by derek-CA. TrustedUserCAKeys configured in sshd_config.
